--------factory------------
factory is a global data base of all tb component 
by using factory we can register object and component
   
   syntax:objection:`uvm_object_utils(class_name)

   syntax:component:`uvm_component_utils(class_name)


  ex:Consider a sugar tablet factory.

The formula for preparing the sugar tablet is very important.

There are two ways to store the formula:

   1. Writing Formula on Paper
     .Every worker must carry the paper
     .If paper is lost, formula is lost
     .Difficult to access everywhere
     .Need manual handling every time
2. Storing Formula in Server
     .Formula stored in central server
      .Any machine can access it anytime
      . No need to carry paper
       .Easy to modify and reuse
       .All departments can use same formula


Relation to UVM Factory
In UVM, factory registration works like the server.
